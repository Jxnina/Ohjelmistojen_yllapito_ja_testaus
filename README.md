# 02_Unit_Testing

JavaScript module for basic arithmetic operations with unit tests using Mocha + Chai (expect).

The module includes add, subtract, multiply and divide functions.

Division by zero throws an error ("ZeroDivision").

## Installation
npm install

## Run main program
npm run start

## Run unit tests
npm test
